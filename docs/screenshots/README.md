# Screenshots

Drop the images referenced by [`documentation.md`](../../documentation.md) here.
Filenames must match exactly so the embeds resolve on GitHub:

| File | Used for | Suggested size |
| :--- | :--- | :--- |
| `app-showcase.png` | Main hero mockup (top of the docs) | ~1720×1080 (renders at 860px wide) |
| `feature-crowd.gif` | MRT/LRT Crowd Status demo | ~1280px wide (renders at 640px) |
| `feature-alerts.gif` | Live Service Alerts demo | ~1280px wide (renders at 640px) |
| `feature-bus.gif` | Bus Arrival Timings demo | ~1280px wide (renders at 640px) |

**Tips**
- Export at 2× the display width for crisp rendering on Retina/HiDPI screens.
- Keep GIFs short (4–8 s) and optimized (e.g. with `gifsicle`) to stay under ~5 MB.
- Capture both light and dark mode if you want to showcase night-time commuting.
